# three-template

Three.js template for Bun with oxlint and oxfmt.

## Setup

```bash
bun install
```

## Development

```bash
bun run dev
```

## Checks

```bash
bun run fmt
bun run lint
bun run typecheck
```

## Versions

- Bun `1.3.11`
- Three.js `0.183.2`
- oxlint `1.59.0`
- oxfmt `0.44.0`
- TypeScript `next`
